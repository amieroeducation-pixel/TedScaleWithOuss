## 📱 Comment Utiliser

### Comparer les variantes

1. Va sur `/sequences-variants`
2. Clique sur "Post-premier contact TNS" dans la sidebar
3. Clique sur "J+0 • EMAIL" pour voir le premier step
4. Utilise les boutons **A, B, C, D, E, F** pour comparer les 6 variantes
5. Lis le preview en bas pour voir le message complet

### Lancer une séquence

1. Va sur `/crm` (Kanban)
2. Trouve un prospect "À contacter"
3. Clique sur les 3 points → "Démarrer séquence"
4. Choisis "Post-premier contact TNS"
5. La séquence démarre automatiquement (emails, WhatsApp, SMS...)

### Modifier un message

1. Va sur `/settings`
2. Onglet "Séquences"
3. Sélectionne la séquence et le step
4. Édite le texte directement
5. Sauvegarde

## 💡 Astuce

**Chaque variante utilise un angle psychologique différent**:
- Cost of inaction (coût de l'inaction)
- Social proof (témoignages)
- Breaking point (point de rupture)
- Timeline du regret ("dans 10 ans...")
- Question piège (révèle un gap)
- Ultimatum doux (dernière chance)

➡️ **Teste plusieurs variantes** pour voir celle qui convertit le mieux!
